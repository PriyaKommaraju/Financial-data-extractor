# Financial Data Extractor

An AI-powered web application that extracts financial metrics such as Revenue and EPS from financial news articles using LangChain, Groq, and Llama 3.3.

## Features
- Extract Revenue (Actual & Expected)
- Extract EPS (Actual & Expected)
- JSON parsing with LangChain
- Streamlit web interface

## Tech Stack
- Python
- Streamlit
- LangChain
- Groq API
- Llama 3.3 70B

## Run Locally

```bash
pip install -r requirements.txt
streamlit run main.py
```
